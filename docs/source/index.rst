Welcome to PlottingApp's documentation!
=======================================

Description:
------------

PlottingApp is a simple application to visualize data from multiple source files format.
It is written in Python using mainly following libraries:
 - matplotlib for plotting data
 - pandas and numpy for reading data
 - PySide2 for GUI

.. warning::

   PlootingApp is still under development. Please report any issue.
